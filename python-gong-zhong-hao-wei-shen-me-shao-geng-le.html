你不是 Python 博主吗？为什么现在少更了

2025-07-14
  • AI

最近时不时会有人私信我，问：

「你不是 Python 公众号博主吗？怎么现在不写了？」
「你写点 Python 的吧」
…

私信背后的惯性提问

最开始看到这种私信，我会下意识地想解释点什么。但后来我发现，这种解释根本没有意义。它们只是出于一种惯性而发出的提问，和真正的关心无关。我后台数据又不是看不到，我写了，你又不看。我不写，你又要问。

我写不写 Python，从来不是因为我背负了什么身份，也不是因为我需要讨好谁。

AI 改变了学习路径

当初写，是因为我确实在用，也愿意分享。现在少写了，是因为这个时代已经变了。尤其是 AI 的出现，几乎重新定义了「怎么学习」这件事。

现在只要你真心想学 Python，完全可以靠 AI 搞定。用对工具、提好问题，它给出的反馈比大多数教程还清晰。甚至你边学边问，边问边改，效率高得离谱。

AI 已经大幅降低了学习成本，它甚至不需要我出现在你面前，你都能学得很好。

这是现实，真正想学的人，不会因为我写不写而决定学不学。

我一直写的不只是 Python

而且说句实话，如果你真的细看我以前写的 Python 教程，就会发现它们从来不只是代码堆砌。那些教程背后，我更在意的是它的创意、它解决问题的方式，还有过程中我观察到的一些细节和思考。换句话说，我一直在写的，从来就不只是「Python」。

而我现在更在意的，不是某个技巧有没有写过，也不是某个技术栈还要不要更新。我更关心我在想什么、我观察到了什么、我正在经历什么样的阶段。我希望我写的内容，能带着点温度、带着点角度去思考。

内容方向应该由创作者决定

不管我写啥，即使我未来如果有写教程类的东西，那一定是因为我真的碰到了什么好玩的东西，有话想说，有思路想分享，而不是因为谁催我、谁提醒我「应该写什么」。

我更新什么内容，是我自己决定的，这点，从一开始就没变过。
