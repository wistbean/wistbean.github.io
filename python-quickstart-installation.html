常见问题

python 命令找不到怎么办？

通常是 PATH 未配置。可重新安装并勾选 PATH，或手动将 Python 安装目录加入系统环境变量。

pip 命令不可用怎么办？

优先使用 python -m pip，稳定性更高。

学习 FAQ

Windows 安装后输入 python 仍然无效怎么办？

先执行 py --version 检查 Python Launcher；若可用，优先用 py。若都不可用，重新安装并勾选 Add Python to PATH。

pip 与 python -m pip 有什么区别？

本质都调用 pip，但 python -m pip 会绑定当前解释器，能减少多版本 Python 下装错环境的问题。

新手安装后第一步应该做什么验证？

先跑 python --version、python -m pip --version，再执行一个 hello.py，确认解释器、包管理和代码执行都正常。
