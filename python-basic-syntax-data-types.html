学习 FAQ（统一）

变量和常量在 Python 里怎么区分？

Python 语法层面没有真正“只读常量”，通常用全大写命名（如 PI = 3.14）表示约定常量。

为什么 1/2 结果是 0.5 而不是 0？

Python3 中 / 是真除法，结果是浮点数；若只要整数部分，用 // 整除。

字符串里反斜杠总报错怎么处理？

路径建议用原始字符串 r"C:\\path"，或将 \ 写成 \\，避免转义冲突。
