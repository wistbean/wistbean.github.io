    SSM视频教程第一节课程概要

    面向的学习人群

    1. 具备有一定的java基础
    2. 初级开发者 -> 中高级开发者
    3. 具备一定的自学能力和执行能力！

    技术选型

    1. Spring
    2. Spring MVC
    3. Mybatis
    4. redis
    5. solor
    6. EasyUI
    7. UEditor
    8. JQuery
    9. Freemark
    10. activMQ
    11. httpClient
    12. MySQL

    开发环境

    1. IntelliJ IDEA
    2. Maven
    3. Tomcat7
    4. JDK
    5. Nginx
    6. Git
    7. postman
    8. sqlyog
    9. win7

    计划

    • 技术架构 （集群和分布式架构的区别）
    • 工程搭建 （maven）
    • SSM框架的整合
    • Mybatis逆向工程以及使用
    • 日志的添加与使用
    • 拦截器
    • 后端功能 （系统的开发，图片系统，数据等等）
    • 前端功能 （商品浏览，下订单，购物车等等。。）
    • redis使用以及集群搭建
    • solor使用和集群搭建
    • JMS 消息队列 （activMQ）
    • sso单点登录
    • restful服务
    • 在Linux上部署
