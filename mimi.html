告诉你一个秘密

2024-09-21

不知道你有没有遇到这样的情况：

有天你突然想起了某位许久未见的朋友，于是你会想起他的模样，想起你们之前谈笑的样子，你心里想见他一下，然后没过两天，你们就在没有任何通知的情况下遇上了。

有天你突然很想吃一顿饺子，于是你会想起它的味道，然后下班回家后发现家里真的做了一顿饺子，而且包的馅也是和你今天所想的一样。

…

反正我都遇到过。

这是我最近看了一本名叫《秘密》的书，才让我回想起了这些看起来近乎神奇的小事。

而这个秘密是：

吸引力法则

可能你也听说过它，简单来说，就是我们一直思考着什么，感受着什么，就会吸引什么样的事物或经历到你的身边。

作者做了一个有意思的比喻，大概是说，就像磁铁的磁场一样，我们的思想也是有能量的。

这种能量不仅影响我们自己的状态，还会在宇宙中发出信号，吸引与我们想法相对应的能量。

比如，当你经常想着成功和富足时，就像磁铁一样，你会吸引更多的成功机会和富足的体验。

思想——变成——实物

好的思想会吸引好的事物，坏的思想则吸引不好的事物，老人言，心态要放好，说的就是这个道理吧。

就像有时候你想拥有某样的东西的时候，你每天都想着它，想着想着就真的有了，它真的会被你吸引到你身边，所谓「信则有，不信则无」。

作者还给出了创造的过程：

1，要求

2，相信

3，接收

首先你要非常清楚你想要什么，然后向宇宙发出请求，然后你要有非常强的信念，相信你现在就已经拥有了，接着你要去感受它，你知道你已经拥有了你想要的，你会去体验这种兴奋或者愉快的感觉。

当你这么做之后，宇宙会以某种形式发送给你，你接收它就好了，不用管宇宙是如何让你得到的，反正就是你吸引来的。

我突然想到了一个类比，想象一下你在写 API 调用服务器的过程。

你在做 API 请求的时候，那就是你的思想：

在编程中，API 请求是由客户端发出的，旨在从服务器获取特定的数据或执行某个操作。同样的，我们的思想就像是向宇宙发出的请求。每个思想都是一个请求，传达了我们希望得到的事物或体验。

就像 API 调用需要明确的参数（请求方法、路径、数据等），思想也需要明确的方向和内容。越明确的思想，就像是一个精确的 API 请求，能够更快更有效地得到想要的响应。

而服务器就是宇宙:

服务器接收到 API 请求后，会根据请求的内容进行处理，返回相应的数据。类似地，宇宙接收到我们的思想请求后，也会根据这些「请求」开始工作，寻找匹配的现实状况来回应我们。

当然，我们并不需要关心它是如何实现的，我们只要相信它会返回数据就好了，宇宙会根据我们的思想和信念返回相应的生活画面或体验。这些返回的「数据」就是我们在现实生活中看到和经历的事物。

响应就是你吸引来的实物：

当服务器返回数据时，它是根据 API 请求的内容来决定的。生活中的回应也是如此。我们想什么、相信什么，就会吸引到相应的结果。积极的思想和信念就像一个正确配置的 API 请求，会引来正面的结果。

另外，作者为了能加强我们的吸引，还给了我们两个「工具」：

1，感恩

2，视觉化

因为当你花时间对你已经拥有的实物表达感恩，以及在想要的事物到来之前对它们进行感谢时，会产生一种强大的情感，而这种情感，能够帮助你保持积极的心态，提升你的频率。

在你的脑海中，尽可能清晰和生动地想象你已经拥有你所渴望的事物。注意细节，感受这件事物已经成为你生活的一部分。不仅要想象它的样子，还要感受拥有它后的喜悦和满足。而这样的视觉化，能够增强你的吸引力，让你更快地与想要的事物频率对齐。

通过这些方法，可以增强我们的吸引力。

可以从小的目标开始去应用你的吸引力，就像作者一样，他每次开车快到达目的地的时候，都会想象等会停车场会有一个空位为他留着，等他到了时候，真的就有位置给他停。

当你熟练应用好吸引力法则后，你可以吸引任何你想要的东西，物质？伴侣？幸福？事业？…请你相信，你就是一个牛逼的存在。

就像雷军所说的：

永远相信美好的事情即将发生！

这就是吸引力法则。

那么写到这里，我也来应用吸引力法则，我现在就向宇宙发起需求，我希望这篇文章能够让 1000 人看到。

好了，我开始相信已经有 1000 人看到这篇文章了，还有人给我点赞，有人转发给他的朋友了，有人看完这篇文章开始改变自己的心态了，甚至去翻这本书了，我还看到我帮到了人了，有人思想开始转变了，整个人变得开心的笑起来了，太好了！很感谢遇到你们！！
