    • bug

Cannot resolve bean ‘dataSource’

在 IDEA 中使用 DataSourceTransactionManager 的属性 dataSource，发现 ref = dataSource 没用引用到，报红色

原因是在没有依赖到 Spring。

解决

  1. 打开 Project Structure

  2. 选择相应的模块，点击上方的 + 号

  3. 选择 Spring

  4. 将配置文件依赖进来

  5. 点击 apply 确认，问题解决
